# From Text to Measure: Supplementary Materials and Appendices

This repository contains the supplementary methodological appendices, implementation materials, and reproducibility resources associated with the Master's Thesis:

**From Text to Measure: A Framework for Psychometric Construct Measurement Using Unstructured Data**

Author: Xiaowei Cai  
Master's Degree in Data Science  
Universitat Oberta de Catalunya (UOC)

---

## Repository Purpose

The repository serves as an online supplementary archive for the TFM manuscript.  
It provides detailed methodological documentation, coding protocols, implementation workflows, and supporting materials that are referenced throughout the dissertation but are not included in full within the main manuscript due to space constraints.

The repository is intended to improve:

- methodological transparency,
- auditability,
- reproducibility,
- implementation traceability, and
- future methodological reuse.

---

### Appendix VII — Corpus-Level Deployment and Score Construction

---

## Important Notes

- The repository is intended primarily as a methodological and implementation archive.
- The trained ACOS prediction model, scoring logic, prompts, and evaluation documentation are included as reusable methodological outputs.
- Some internal evaluation resources used for calibration and held-out assessment are documented descriptively in the manuscript and appendices but are not distributed as public benchmark datasets.
- The repository is provided for academic transparency and research reproducibility purposes.
